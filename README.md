Hello!

I'm Naren Ratan, keen microtonalist and computer programmer.

If you're interested in microtonal music, you might like:
- [scale-library](https://scalelibrary.org), a library of microtonal scales with sources attributed ([repo](https://github.com/narenratan/scale-library))
- [Another look at Wilson's keyboard mapping system](https://www.xenharmonikon.org/2026/05/19/another-look-at-wilsons-keyboard-mapping-system/), my Xenharmonikon article on the maths behind Erv Wilson's keyboard mappings ([repo](https://github.com/narenratan/gral))

If you like Forth and APL, you might get a kick out of
[jonesforth_arm64_apl](https://github.com/narenratan/jonesforth_arm64_apl), an
ARM version of the wonderful JonesForth. It uses APL symbols and does some cool
stuff like using NEON registers to parse hex numbers.

I work as a quant dev, specifically on optimization problems. I like staring at
solver logs and finding ways to solve big models quickly (or at all).
